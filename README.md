# Aman shop

**Aman shop** is a feature-rich website built with Next.js, Tailwind CSS, Sanity, and Stripe integration. This fully responsive site offers a seamless user experience with both dark and light themes, allowing users to choose their preferred mode. It boasts robust functionality, including a powerful product search, filtering, and ordering system, ensuring that users can easily find and purchase the items they desire. The integrated shopping cart feature facilitates smooth transactions, while Stripe ensures secure and hassle-free payment processing.
